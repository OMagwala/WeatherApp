OVERVIEW 
*This is a simple weather web application that allows a user to check the weather conditions of a location.
*It uses the OpenWeatherMap API to retrieve data.

FEATURES 
*Displays the current temperature, weather description and detailed weather conditions for the searched location.
*Has a search functionality to find weather conditions for different locations.

USAGE 
*Enter the name of a location in the search bar.
*Click the search button to get the current weather information for the specified location.
*View the displayed weather details, including temperature, conditions, and additional information.
